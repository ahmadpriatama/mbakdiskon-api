Menggunakan API Deal Aktif

Fungsi API ini adalah untuk mendapatkan deal-deal yang sedang aktif

URL Endpoint :

http://mbakdiskon.com/api/deals/

METODE API :

GET

Parameter API :

  • location : Topic stream
    • Harus selalu disertakan
    • value yang tersedia adalah: 1(Yogyakarta), 2(Semarang)
  • language : pageing start
    • Harus selalu disertakan
    • value yang tersedia adalah: id_id
  • rt : return type (well formed xml)

Example :

http://mbakdiskon.com/api/deals/location/1/language/id_id
