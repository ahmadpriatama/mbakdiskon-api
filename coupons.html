Menggunakan API Coupons

Fungsi API ini adalah untuk mendapatkan list coupon yang dimiliki

login required

URL Endpoint :

http://mbakdiskon.com/api/coupons/

METODE API :

GET

Parameter API :

  • language :
    • Harus selalu disertakan
    • value yang tersedia adalah: id_id
  • rt : return type (well formed xml)

Example :

http://mbakdiskon.com/api/coupons/language/id_id/
