Menggunakan API Info Coupon

Fungsi API ini adalah untuk mendapatkan info lengkap dari sebuah coupon

login required

URL Endpoint :

http://mbakdiskon.com/api/coupon/

METODE API :

GET

Parameter API :

  • code :
    • Harus selalu disertakan
    • value yang tersedia didapat dari API Coupons
  • rt : return type (well formed xml)

Example :

http://mbakdiskon.com/api/coupon/code/1241930804

Result : 

<couponic>
    <status>true</status>
    <errorCode>0</errorCode>
    <errorMessage/>
    <coupon>
        <id>8518</id>
        <hashid>#182-11888-B82555DBAF7E</hashid>
        <deal>[Special Offer] 30% OFF: Waffle, Snack, and Beverage Packages from Waffle Days | Saturday Waffle+French Fries+Hot Cappuccino/Hot Chocolate now only Rp 20.000 (Normal Rp 29.000)</deal>
        <name>ahmad p.</name>
        <firstname>ahmad</firstname>
        <lastname>priatama</lastname>
        <status>available</status>
        <code>1241930804</code>
    </coupon>
</couponic>
