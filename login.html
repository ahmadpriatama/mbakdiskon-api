Menggunakan API Login

Fungsi API ini adalah untuk melakukan login ke sistem. dibutuhkan sebelum memanggil API coupon

URL Endpoint :

http://mbakdiskon.com/api/login/

METODE API :

POST

Content type: application/x-www-form-urlencoded

Parameter API :

  • email :
    • Harus selalu disertakan
    • email user
  • password :
    • Harus selalu disertakan
    • password user
  • rt : return type (well formed xml), yang merupakan session id

Example :

http://mbakdiskon.com/api/login

Result : 

<couponic>
    <status>true</status>
    <errorCode>0</errorCode>
    <errorMessage/>
    <session_name>PHPSESSID</session_name>
    <session>rmpnk70og5g8cfrua2na3qf5v1</session>
</couponic>
